# 💰 Bolso · Gestão Financeira Pessoal

O **Bolso** é uma aplicação web de controle financeiro desenvolvida para ajudar você a organizar suas finanças de forma simples, inteligente e visual. O app conta com suporte a múltiplas contas bancárias, controle de lançamentos, porquinhos de reserva e monitoramento de gastos fixos.
O app foi planejado a partir de uma dificuldade pessoal para organizar planilhas de uma forma simples, e como eu estava na faculdade, criei um app para me organizar financeiramente e postei para que talvez alguem que também tenha dificuldade de organizar suas proprias planilhas, conseguisse finalmente entender pra onde vai o dinheiro dela
A função de "Reservas", baseia-se em uma reserva imaginaria do saldo do seu dinheiro, então se você tem 1000 na conta, mas tem 500 reservados nessa conta para alguma despesa fixa, ele automaticamente avisa você o quanto você pode gastar, para não utilizar o dinheiro que esta reservado

---

## 🚀 Funcionalidades Principais

* ** Múltiplas Contas / Bancos:** Cadastre diferentes contas (ex: Nubank, Carteira, Itaú) com saldos iniciais distintos e alterne entre elas usando abas dinâmicas.
* ** Lançamentos Filtrados:** Adicione ganhos e despesas que ficam automaticamente vinculados ao banco selecionado.
* ** Reservas Financeiras (Porquinhos):** Separe uma quantia para objetivos específicos e adicione ou retire valores diretamente no card da reserva, como ja explicado anteriormente.
* ** Controle de Gastos Fixos:** Monitore suas contas recorrentes (aluguel, internet, etc.) e alterne o status entre "Pendente" e "Pago".
* ** Inteligência nos Visores:** * *Pode Gastar:* Mostra o saldo total acumulado de todos os bancos subtraindo o valor que está guardado nas suas reservas.
    * *Saldo Total:* Soma em tempo real de todo o dinheiro de todas as suas contas.
    * *Gasto no Mês:* Mostra o quanto você já gastou especificamente no banco selecionado.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero utilizando tecnologias web nativas para treinar conceitos de JavaScript:

* **HTML5:** Estruturação semântica da aplicação e dos modais.
* **CSS3:** Layout moderno baseado em Grid e Flexbox, tipografia integrada e transições suaves.
* **JavaScript (ES6+):** Lógica de negócios, manipulação dinâmica do DOM e escopo global estruturado.
* **LocalStorage:** Persistência de dados local no navegador, permitindo que seus dados fiquem salvos, por enquanto, na sua maquina.
